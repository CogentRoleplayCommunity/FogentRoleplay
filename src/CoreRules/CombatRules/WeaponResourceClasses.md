# Weapon Resource Classes

Weapon Resource Class determines the resource a weapon uses to opperate.

- Bolt
- Arrow
- Blowgun Dart
- Sling Bullet
- 9mm Ammo
- 5.56 Ammo
- .308 Ammo
- RPG Ammo
