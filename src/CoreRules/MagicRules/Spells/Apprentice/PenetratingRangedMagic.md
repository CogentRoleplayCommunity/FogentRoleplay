# Penetrating Ranged Magic

Character focuses their magic on a single penetrating attack.

## Info

**Duration:** Instantaneous

**Range:**

| Skill level |  Range Type  |
| :---------: | :----------: |
|      1      |   Extended   |
|      2      |     Long     |
|      3      | Sharpshooter |
|      4      |   Extreme   |

**Engageable Opponents:** Focused

**Penetration:** 3

**Magic Resource Required:** 1

**Area Of Effect:** None

**Skills required:**

- Any magic (level 1)
